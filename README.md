# Student Placement Portal

A full-stack web application for managing student placements. This project helps students register, view available job openings, apply for jobs, and track their application status.

## Features

- Student registration
- Student login
- Add job openings
- View available jobs
- Apply for jobs
- View applications
- Update application status

## Technologies Used

### Frontend
- React.js
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Tools
- VS Code
- Postman
- Git
- GitHub
